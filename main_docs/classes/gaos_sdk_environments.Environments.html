getEnvironmentFiles

  • getEnvironmentFiles(
        environment: string,
        path: string,
        params?: GetEnvironmentFilesParams,
        options?: Omit<RequestOptions, "extra_body">,
    ): APIPromise<GetEnvironmentFilesResponse>

    Retrieves a file or directory from an environment's snapshot.

    If the path points to a directory:

    • If alt=media is specified, it downloads the directory as a tarball.
    • Otherwise, it returns the metadata of the files and directories immediately within it (directory listing). If the path points to a file:
    • If alt=media is specified, it downloads the file.
    • Otherwise, it returns the file metadata.
